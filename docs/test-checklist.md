# テストチェックリスト

## 1. フロントエンド UI

### 初期表示
- [ ] 現在時刻がリアルタイムで更新される
- [ ] 出勤ボタンが有効（緑色）で表示される
- [ ] 退勤ボタンが無効（グレー）で表示される
- [ ] 課題報告の送信ボタンが無効（テキストエリアが空）で表示される

### 出勤打刻
- [ ] 出勤ボタンを押すと確認ダイアログが表示される
- [ ] キャンセルを押すと何も変わらない
- [ ] OK を押すと出勤時刻が表示される
- [ ] 出勤ボタンが無効になる
- [ ] 退勤ボタンが有効（赤色）になる
- [ ] スプレッドシートに出勤時刻が記録される
- [ ] LINE に出勤通知が届く

### 退勤打刻
- [ ] 退勤ボタンを押すと確認ダイアログが表示される
- [ ] OK を押すと退勤時刻と勤務時間が表示される
- [ ] 退勤ボタンが無効になる
- [ ] スプレッドシートに退勤時刻と勤務時間（分）が記録される
- [ ] LINE に退勤通知が届く

### 課題報告
- [ ] テキストエリアに入力すると送信ボタンが有効になる
- [ ] テキストエリアを空にすると送信ボタンが無効になる
- [ ] 送信ボタンを押すと課題報告シートに記録される
- [ ] 送信成功後にテキストエリアがクリアされる
- [ ] LINE に課題報告通知が届く

---

## 2. エラーケース

### バリデーション
- [ ] 課題報告テキストが空のまま送信しようとするとエラートーストが表示される
- [ ] 出勤済みの状態で出勤ボタンが押せないことを確認する

### 通信エラー（開発者ツールでオフラインに設定して確認）
- [ ] ネットワーク切断時に打刻するとエラートーストが表示される
- [ ] エラートーストが 3 秒後に自動で消える

---

## 3. PWA

- [ ] `manifest.webmanifest` の内容が正しく認識される（Chrome DevTools > Application > Manifest）
- [ ] Service Worker が登録されている（Chrome DevTools > Application > Service Workers）
- [ ] スマホのブラウザで「ホーム画面に追加」ができる
- [ ] ホーム画面のアイコンが設定したアイコン画像になっている
- [ ] ホーム画面から起動するとスタンドアロンモードで表示される（ブラウザの UI バーが消える）

---

## 4. localStorage による状態復元

- [ ] 出勤打刻後にページをリロードしても出勤時刻が表示されたままになる
- [ ] 出勤打刻後にページをリロードしても出勤ボタンが無効のままになる
- [ ] 日付をまたいだ場合（手動でlocalStorageのdateを書き換えて確認）に状態がリセットされる
